## Projetos em Destaque

### 🔎 [Detecção de Fraudes Financeiras com ML](https://github.com/ricardohenrique1609/Fraude-detection)
Modelo de classificação para identificar transações financeiras fraudulentas em datasets altamente desbalanceados.

- **Técnicas:** SMOTE, XGBoost, Scikit-learn pipelines, threshold optimization
- **Métricas:** AUC-ROC, Precision-Recall, F1-Score
- **Foco:** Feature engineering comportamental — velocidade transacional, padrões temporais e anomalias estatísticas
- **Stack:** Python · Scikit-learn · XGBoost · Pandas · Matplotlib

---

### 🚌 [Previsão de Destinos — ClickBus](https://github.com/ricardohenrique1609)

- **Objetivo:** Personalização e antecipação de demanda
- **Pipeline:** EDA → Feature Engineering → Modelagem → Avaliação de métricas de negócio
- **Stack:** Python · Scikit-learn · Pandas · Seaborn
